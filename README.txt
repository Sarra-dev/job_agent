This is a Telegram bot that allows users to upload their CV (PDF, DOCX, TXT, DOC, or image formats like JPG/PNG). The bot extracts information from the CV, stores it, and helps the user find job offers in France or Germany based on their preferences.

Features

Accepts CV uploads in PDF, DOCX, DOC, TXT, JPG, PNG, BMP, TIFF.

Extracts text using OCR for images.

Uses spaCy for NLP tasks.

Matches user profile to available job offers using the Adzuna API.

Conversational flow to collect user preferences (job title, country, city, level).

Requirements

See requirements.txt for Python dependencies.
